# 🍕 Pizza Sales Analysis Using SQL

## 📌 Project Overview
This project analyzes pizza sales data using **MySQL** to identify customer purchasing patterns, sales performance, and revenue trends. By querying transactional data, the project generates key business insights that can support data-driven decision-making.



## 🎯 Objectives
- Analyze overall sales performance.
- Calculate key business KPIs.
- Identify the best-selling pizzas.
- Analyze category-wise sales contribution.
- Understand customer ordering preferences.
- Generate business recommendations based on the findings.



## 🛠 Tools & Technologies
- MySQL
- SQL
- Microsoft PowerPoint
- GitHub



## 📂 Dataset
The project uses four relational tables:

| Table | Description |
|--------|-------------|
| Orders | Order ID, Date, Time |
| Order Details | Order ID, Pizza ID, Quantity |
| Pizzas | Pizza ID, Size, Price |
| Pizza Types | Pizza Name, Category |



## 🧠 SQL Concepts Used
- SELECT Statements
- INNER JOIN
- Aggregate Functions (SUM, COUNT, AVG)
- GROUP BY
- ORDER BY
- LIMIT



## 📊 Key Performance Indicators (KPIs)

| KPI | Result |
|------|---------|
| Total Orders | **21,350** |
| Total Revenue | **$817,860** |
| Most Ordered Pizza Size | **Large** |
| Best Performing Category | **Classic** |



## 🔍 Business Questions Answered
- How many total orders were placed?
- What was the total revenue generated?
- Which pizza size was ordered the most?
- Which pizzas were the top sellers?
- Which pizza category generated the highest sales?
- What percentage did each category contribute to total sales?



## 💡 Key Insights
- Processed **21,350 customer orders** using SQL.
- Generated total sales of **$817,860**.
- Large pizzas were the most preferred size among customers.
- The Classic category contributed the highest sales volume.
- Identified the top-performing pizzas based on quantity sold.
- Category-wise analysis revealed customer purchasing preferences.



## 📈 Business Recommendations
- Maintain higher inventory for top-selling pizzas.
- Introduce combo offers around best-selling products.
- Promote low-performing pizzas using discounts.
- Focus marketing campaigns on high-performing categories.
- Use sales trends to improve inventory planning.
